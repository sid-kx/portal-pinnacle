Broker of Record Guide

How to manage the portal
1

Review requests. Open Agent Approvals to see pending access requests from agents who submitted the request form.

2

Approve or reject. Approved agents can log into the portal. Rejected requests stay blocked from portal access.

3

Manage active agents. Use the Agents page to view active accounts. Deleting an agent disables their login and removes their public profile.

4

Control website content. Use New Construction to manage projects that will appear on the main Pinnacle Realty website.
